# 指令说明

1. 站立模式

     GaitMode  =  GaitMode_Stand;

 - **发送数据为     ？？**
 - 初始化后即为站立模式
 - 误按其他未定义按键时也会恢复站立模式

2. Trot步态模式

    GaitMode  =  GaitMode_Trot;

 - **发送数据为 11**
 - 按下**Trot步态**按键后，进入Trot步态模式
 - 在此模式下，有7个子模式可供选择
    1. **前进**：发送数据为02
    2. **后退**：发送数据为08
    3. **向左平移**：发送数据为04
    4. **向右平移**：发送数据为06
    5. **原地踏步**：发送数据为05
    6. **向左旋转**：发送数据为01
    7. **向右旋转**：发送数据为03
    8. **自动避障**：发送数据为19

3. 扭动模式

    GaitMode  =  GaitMode_Body_Twist;
    
 - **发送数据为13**
 - 按下**身体扭动**按键后，进入扭动模式
 - 在此模式下，有7个子模式可供选择
    1. **向前扭**：发送数据为02
    2. **向后扭**：发送数据为08
    3. **向左扭**：发送数据为04
    4. **向右扭**：发送数据为06
    5. **身体向下**：发送数据为05
    6. **向左摆**：发送数据为01
    7. **向右摆**：发送数据为03
    8. **IMU自稳**：发送数据为19
